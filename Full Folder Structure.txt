## Key Features ##

Control Bar (All Pages):
- Time display with day name + 24-hour format (updates every 1 second)
- Dark mode toggle button (persists in localStorage)
- Sound toggle button (persists in localStorage)
- Home button on test pages (centered, shows warning modal)
- Go-to-top button on home page (centered, smooth scroll)

Navigation Warnings:
- Warning modal for refresh (F5, Ctrl+R, Cmd+R, browser refresh)
- Warning modal for back button press
- Warning modal for home button click
- Warning modal for header/title click (on test pages)
- Custom modal only (no browser default warnings)

Footer:
- Compact footer with creator credits
- "Download Android App" plain text link below credits
- Smart detection: Play Store on Android, APK download on others

Responsive Design:
- Mobile breakpoints: 768px, 480px
- Control bar button sizes scale appropriately
- All features work on mobile devices
